Key Discoveries

1. Momersion Indicator (Published Formula)

The Momersion indicator was published by PriceActionLab and is central to the MRMOM strategy:

Momersion(n) = 100 * MOMc / (MOMc + MRc)

Where:
  MOMc = count of same-sign consecutive returns (momentum)
  MRc  = count of opposite-sign consecutive returns (mean-reversion)
  n    = lookback period (typically 250 days)

For each period from 1 to n:
  If r(i) * r(i-1) > 0 --> increment MOMc (momentum)
  If r(i) * r(i-1) < 0 --> increment MRc (mean-reversion)

Interpretation:

  • Above 50%: Momentum-dominated regime - use trend-following
  • Below 50%: Mean-reversion-dominated regime - buy dips

Source: Momersion Indicator - PriceActionLab

2. RSI(2) Mean-Reversion Pattern

Based on Larry Connors' research, likely used in MRETF, ETFMR strategies:

Entry:  RSI(2) < 10  (deeply oversold)
Exit:   Price crosses above 5-day SMA
Filter: Price > 200-day SMA (uptrend only)

Key Statistics (from Connors research):
  - Win rate: ~69%
  - Avg holding period: 4-5 days
  - No stop-loss (destroys profitability)

Source: RSI(2) - StockCharts

3. B2S2 Pattern (Buy 2 Sell 2)

PriceActionLab describes B2S2 as having "no parameters to optimize and no filters":

Entry: 2+ consecutive down days
Exit:  2+ consecutive up days

For multi-stock version (B2S2DJ):
  - Rank by lowest rate-of-change in lookback
  - Maximum 10 concurrent positions
  - Equal-weight allocation
  - Position size: Available equity / 10

4. Seasonality Patterns

ETFSEAS exploits well-documented calendar effects:

  • Turn-of-month effect: Last trading day + first 3 days of new month
  • End-of-month effect: Last 4 trading days tend to be positive
  • Day-of-week: Monday/Friday often show positive bias

Very low exposure (~28%) suggests trading only on specific days matching these patterns.

5. Trend-Following (TFDLS)

Classic Donchian-style breakout system:

Long Entry:  Price > N-day high (breakout)
Short Entry: Price < N-day low (breakdown)
Exit:        Opposite breakout or ATR-based stop

Configuration:
  - 23 futures contracts (diversified)
  - Identical parameters across all markets
  - ATR-based position sizing
  - Capital requirement: ~$1M for proper sizing

Key characteristic: Negative correlation with SPY (-0.07) provides crisis alpha.

Key Insights for Practitioners

What Works

  • Low beta strategies: MRETF (0.14), ETFSEAS (0.10) achieve returns with minimal market exposure
  • Regime adaptation: MRMOM's Sharpe of 1.15 suggests regime-switching adds significant value
  • Simple rules: B2S2's "no parameters" approach suggests robustness over optimization
  • Diversification: TFDLS's negative correlation provides portfolio-level benefits

Risk Factors

  • No stop-losses: Most mean-reversion strategies don't use stops - high win rate compensates
  • Regime dependency: Strategies can fail when market regimes change
  • Transaction costs: High-turnover strategies are sensitive to slippage (2-5% impact)
  • Data requirements: Stock strategies need delisted data (Norgate), futures need continuous contracts

Implementation Recommendations

  1. Start with MRETF approach: RSI(2) with trend filter is well-documented and simple
  2. Implement Momersion indicator: Use for regime detection to switch between MR and momentum
  3. Test B2S2 variants: Consecutive day patterns are robust across markets
  4. Calendar effects: End-of-month and turn-of-month are well-researched anomalies
  5. Proper data: Use Norgate for stocks (survivor-bias-free), proper continuous contracts for futures
