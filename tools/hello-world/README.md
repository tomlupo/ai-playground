# Hello World Tool

## Purpose

Demonstrates the ai-playground tool pattern:
- PEP 723 inline script metadata
- Rich console output
- Saving outputs to `outputs/` directory

## Usage

```bash
uv run tools/hello-world/main.py
```

## Output

- Prints a formatted table to console
- Saves a markdown file to `outputs/hello-world_{timestamp}.md`

## Notes

Use this as a template when creating new tools.
